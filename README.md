# Kenny De Pauw Portfolio

Personal portfolio built with native HTML, CSS, and JavaScript.

## Local review

Open `index.html` directly in a browser. No build step or local server is required.

## Structure

```text
kennydp.github.io/
├── index.html
├── css/styles.css
├── js/main.js
├── assets/resume.pdf
├── assets/profile.jpeg
└── assets/favicon.svg
```

## GitHub Pages

This is a static site and can be deployed from the repository root.

1. Create a GitHub repository named `kennydp.github.io`.
2. Push `main`.
3. Enable Pages: Settings -> Pages -> Deploy from branch `main` / root.

The published site will be available at `https://kennydp.github.io/`.
